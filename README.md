Welcome to all!

This is a simple local server runned with python http.server, easy to use and minimalist look.

You can start your own local server, for web developing studio, or IoT projects in prototype fase.

The functions are easy to read, he have the start button server, stop server, and wouold you prefier customize the name or port

or directory destination files.

#====== = =
# - FEATURES
#===========

Every event on the server are logged in the database sqlite3,
and their logs loaded from database sqlite3 and
show you them in the black screen, you can clear the list
of logs from the pulisci logs button.

This tool it's very fast to use, plug and play test your projects, 
start the server with easy way to test a static website or html page,
also you can customize the identity of your service with the Setting menu 
in the file label on the top menu.
